# EduPilot AI

An AI-powered educational platform that helps students with notes, study planning, career guidance, and more.

## Features

- **Dashboard**: Overview of user activities
- **Notes**: Manage and organize study notes
- **Study Planner**: Schedule study sessions
- **Career Predictor**: Get career recommendations based on interests and skills
- **AI Assistant**: Chat with an AI for help with studies

## Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express + MySQL
- AI Services: Python + Transformers
- Database: MySQL
